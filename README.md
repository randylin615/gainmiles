# gainmiles

## 環境設置
請確保已安裝 Docker 和 Docker Compose。
最終測試過程有遇到初次使用```docker-compose up```會失敗，
但再次啟動即可解決！

### 1. 取得相關檔案

```bash
git clone https://github.com/randylin615/gainmiles.git
```


### 2. 啟動服務

```bash
cd .\gainmiles\
docker-compose up
```


### 3. API測試
瀏覽器開啟
```bash
http://127.0.0.1:8000/docs
```
